Prepare an Integration Server

  1. Assuming you are using a Windows platform, open an App Connect Enterprise Command Console and run the following command:
    mqsicreateworkdir C:\MyServer
    where C:\MyServer will the working directory for your Integration Server.
  2. Right-Click on the Policy Project in the Application Development tab and create a new Policy. Call it ActivityLogPolicy and click 'Finish'
  3. Open the new policy for configuring and change the Type to Activity Log. After a brief moment the table below should populate with various fields. Find the 'File name' field and set the value to C:\MyServer\activityLog.txt
  4. Ctrl+s to save the Policy and then create a Bar file in the Application Development tab by right clicking and selecting 'BAR file'. Call it MyBar and click 'Finish' it.
  5. To build the newly created Policy into the BAR file:
    - Select the Applications radio button and tick the 'ExampleApp' application
    - Select the Policies radio button and do the same for the 'Policies' Policy Project
    - Click 'Build and Save' to generate the BAR file.
  6. Start up your Integration Server by typing the following command into the ACE Command Console:
    IntegrationServer --name MyServer --work-dir C:\MyServer
  7. Deploy the MyBar.bar file to your Integration Server after making a connection to it within the toolkit.

Deploy

Follow these steps to complete the tutorial

Follow the following steps to complete the tutorial.

  1. Navigate to a Command Prompt and run this command a handful of times:
    curl -X POST http://localhost:7800/request -d HelloWorld
  2. Navigate to your server's working directory:
    C:\MyServer in this example.
  3. You should see a text file with the same name you specified in the ActivityLogPolicy, activityLog.txt in this example.
  4. Open this file in your preferred text editor and you should see various logs with fields seperated by commas. Each log corresponds to your curl requests made previously.
  5. As you can see the Policy we've created and deployed throughout this tutorial has added some useful functionality to our Integration Server, one of many more sophisticated possibilities provided by ACE.
